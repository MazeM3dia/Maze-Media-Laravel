# Maze Media Laravel Site Project

Initial project setup for Maze Media — built on the Laravel Livewire starter kit.

---

| Laravel Fortify (Auth) | ^1.34 |
| Laravel Tinker | ^3.0 |

---

## Getting Started

```bash
composer run setup
composer run dev
```

`composer run dev` starts the Laravel server, queue worker, and Vite dev server concurrently.

---

## Testing

